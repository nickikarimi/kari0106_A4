1) Short Description
Assignment4 is an engaging Hangman Game implemented in Python. It brings the classic word-guessing game to life, where players attempt to discover a hidden word by suggesting letters. The game utilizes the random module to select a word from a predefined list, providing an element of unpredictability and replayability. With each correct guess, the revealed letters contribute to solving the mystery word, creating an enjoyable and challenging experience.

2)How to run the game
 - Access the GitHub repository at
[https://github.com/nickikarimi/kari0106_A4.git]
   - Click on the "Code" button and select "Download ZIP."
   - Extract the downloaded ZIP file to a location of your choice.
   - Navigate to the Project Directory:
     - Open a terminal or command prompt.
     - Use the `cd` command to navigate to the directory where you extracted the repository. For example:
       ```bash
       cd path/to/extracted/folder/kari0106_A4
       ```
   - Run the Python Project:
     - Make sure you have Python installed on your machine.
     - Run the Python project using the following command:
       ```bash
       python main.py
       ```
     - Follow on-screen prompts for an interactive word-guessing adventure.
     - If needed, ensure dependencies are installed:
       ```bash
       pip install -r requirements.txt
       ```
   - Interact with the Hangman Game:
- Start the Hangman Game to reveal a hidden word represented by underscores.
- Guess a letter, and the game will promptly indicate if you guessed correctly or not.
- Continue guessing letters until you successfully decipher the entire word.
- Experience an engaging and interactive hangman challenge with each guess, testing your word-solving skills.
